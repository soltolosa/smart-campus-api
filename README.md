# smart-campus-api

## Overview of my API:
For this coursework I have implemented a RESTful Smart Campus API with the JAX-RS framework
as required by the specifications. This API was designed to manage rooms and the variety of sensors they may contain while maintaining a historical record of sensor readings, updating the currentValue field in the Sensor object when a new reading is recorded, ensuring consistency across the system.
The API also implements a root Discovery endpoint at GET/api/v1, this returns a JSON response that returns basic information and some HATEOAS links as per the requirements. These include rooms, sensors and sensor readings.
